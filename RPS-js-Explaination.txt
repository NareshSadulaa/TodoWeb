const gameContainer = document.querySelector(".container"),
      userResult = document.querySelector(".user_result img"),
      cpuResult = document.querySelector(".cpu_result img"),
      result = document.querySelector(".result"),
      optionImages = document.querySelectorAll(".option_image"),
      restartBtn = document.querySelector(".restart-btn");

Explaination:-
Grabs elements from the HTML so you can work with them:-
1) gameContainer = the game box. (play area of the game).
2) userResult = image showing user's selected option. ( user select chesukunna image display avthadi).
3) cpuResult = image showing computer's option. (Random image of computer).
4) result = result text ("Let's Play", "You Won!", etc.)
5) optionImages = all three clickable options (rock, paper, scissors). 
6) restartBtn = the restart button.
